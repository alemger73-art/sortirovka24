# Project Summary
The project is a digital community portal for the Sortirovka district in Karaganda, Kazakhstan. This "super-app" enhances local engagement and connectivity among residents by providing a platform that includes professional networking, job postings, news sharing, complaint submissions, food ordering, real estate listings, and access to local resources. The platform supports both Russian and Kazakh languages, prioritizes user security with advanced authentication methods, and features a detailed admin panel for effective management. Recent updates have improved the frontend data loading process and enhanced resilience against transient infrastructure errors.

# Project Module Description
The project is structured into several functional modules:
- **Masters**: Connect with local professionals and submit multimedia service requests.
- **Complaints**: Submit and track multimedia-supported complaints.
- **Announcements**: Post and moderate local announcements.
- **Jobs**: Feature for posting local job opportunities.
- **Questions**: Forum for residents to ask and answer questions.
- **Directory**: Comprehensive listing of local services and resources.
- **Categories**: Manage content categories.
- **Admin Panel**: Content management and user oversight, requiring authentication.
- **Telegram Integration**: Notification system for various actions.
- **Image and Video Upload Enhancements**: Improved multimedia upload design with URL and drag & drop support.
- **Banners and News Management**: Manage banners and news articles.
- **Real Estate**: Module for real estate listings with modern design and search capabilities.
- **Food Delivery**: Management of food delivery services, including a park ordering feature and courier panel.
- **Inspectors**: Module for finding local inspectors via an interactive map.
- **New Admin Dashboard**: Overview of activities and statistics.
- **API Client Enhancements**: Improved resilience for DNS/network resolution errors.
- **Error Handling Improvements**: Enhanced retry mechanisms for API calls.
- **Theme Toggle**: Switch between light and dark modes.
- **Progressive Web App (PWA)**: Full offline capabilities and performance improvements.
- **Prefetching Utility**: Preload data for anticipated screens and images.
- **POS Integration**: Updated FrontPad integration module.
- **Business Page**: Module for businesses to register and connect with the community.
- **Account Settings**: Section in the admin panel for changing login credentials.
- **Weather Block**: Displays real-time weather data for Karaganda.
- **Transport Module**: New functionalities for managing bus routes, stops, and notifications.
- **Public Image Download Endpoint**: New endpoint for public access to images without authentication.
- **Food Section Enhancements**: Redesigned UI for food ordering, including delivery zone management and improved UX.
- **Admin Authentication**: Enhanced security with JWT-based stateless authentication, multi-password support, and brute-force protection.
- **Food Park Module**: New geolocation features for food delivery, allowing users to set their location for orders.
- **Admin Password Reset**: A new script to force-reset the admin password to ensure access.
- **Interactive Park Map**: Replaced Leaflet map with an SVG-based interactive park scheme for delivery.
- **Admin Sidebar Redesign**: Restructured navigation for improved usability.
- **Synchronous Image Handling**: Introduction of `StorageImg` component for reliable image rendering across public-facing pages.
- **API Schema Updates**: Added missing fields to API response models for inspectors, masters, announcements, jobs, and banners.
- **SDK Compatibility Middleware**: New middleware to support the Atoms Cloud SDK's data format for API interactions.
- **Database Migration**: Created 29 new tables in Atoms Cloud for various functionalities.

# Technology Stack
- **Frontend**: React 18, TypeScript, Vite, Tailwind CSS, Shadcn/UI, Lucide Icons.
- **Backend**: FastAPI (Python), SQLAlchemy ORM, Alembic for migrations.
- **Database**: PostgreSQL (managed via Atoms Cloud).
- **SDK**: `@metagptx/web-sdk` for API interactions.
- **Deployment**: Atoms Cloud (serverless backend + managed DB).

# Usage
To set up the project:
1. **Install dependencies**:
   ```bash
   cd /workspace/app/frontend
   pnpm install
   ```
2. **Build the project**:
   ```bash
   pnpm run build
   ```
3. **Run the application**:
   Use the provided script to start the application.
